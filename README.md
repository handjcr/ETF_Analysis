# ETF_Analysis
Simple ETF Analysis Tool
Derived from the following prompt:
"I feel like a lot of ETFs these days have a lot of the same funds.  I want to compare ETFs and the assets in which they include to one another to ensure I have minimal overlap between investment opportunities.  Id also like to compare the ETF's rate of return with the individual assets within each of the ETFs.  Can you suggest a few ways to pull together that data and do this analysis?"
